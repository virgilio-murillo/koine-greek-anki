# Card Quality Standards & User Preferences

## Non-Negotiable Rules
1. **Cards must be 100% self-contained** — the user should NEVER need to look anything up elsewhere
2. **Every morpheme must be explained** — prefix, root, AND suffix. If a suffix like -τός appears, explain what it does and give examples
3. **When a root changes form** (βάλλω→βολ-), explain WHY and show the connection
4. **When etymology is opaque** (ἐπιτρέπω = "turn over" → permit), explain the semantic chain step by step
5. **Mnemonics must tell a story**, not just translate components. "quemar-completamente" is USELESS. Include biblical context, Spanish/English cognates, vivid imagery
6. **Include Strong's data** for the compound word AND each component — this provides the "turn over (transfer)" type explanations that make opaque words understandable

## Language
- All explanations in **Spanish**
- User is a native Spanish speaker who also speaks English
- Use Spanish cognates when available (catálisis, apóstol, evangelio, etc.)
- Use English cognates too when helpful (apocalypse = revelation)

## Terminology — Keep It Simple
| ❌ Don't use | ✅ Use instead |
|---|---|
| grado-o, grado-e, grado-cero | "la raíz cambia de X a Y en sustantivos" |
| ablaut | "cambio de vocal en la raíz (patrón regular)" |
| PIE reconstruction | (never mention) |
| reduplicación | "el τι- al inicio de τίθημι es repetición, no parte de la raíz" |
| voz media | "forma reflexiva" or just skip |

## Card Design
- Front: Greek word (large) + "¿De qué se compone esta palabra?"
- Back sections (in order):
  1. Word (36px, dark blue) + meaning (22px, green) + ref + Strong's #
  2. 📦 Componentes — each in a white box with Strong's definition
  3. 🔤 Sufijo — purple box (only when applicable)
  4. 🔀 Cambio de raíz — orange box (only when applicable)
  5. 💡 Mnemotecnia — green box, the main learning aid
